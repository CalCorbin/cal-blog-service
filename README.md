## Local Development

### Prerequisites
- Go 1.24.0 or higher

### Installation and Running

1. Clone the repository
   ```bash
   git clone https://github.com/CalCorbin/cal-blog-service.git
   cd cal-blog-service
   ```
2. Install dependencies
   ```bash
   go mod download
   ```
3Run the application
   ```bash
   go run main.go
   ```
